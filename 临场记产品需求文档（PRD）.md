# 产品需求文档（PRD）
## 产品名称：Geofence Reminder（临场记）
### 版本：V1.0 - Android MVP

---

## 一、产品概述

### 1.1 产品定位
一款基于地理围栏的智能提醒工具，帮助用户在到达特定地点时自动弹出预设提醒，解决「路过却忘了」的日常痛点。

### 1.2 核心价值
- **被动触发，无需记忆**：到达地点自动提醒，不用主动打开App
- **简单直观**：在地图上画个圈，写下提醒内容，搞定
- **轻量纯粹**：不做日历、不做待办清单，只做好「位置→提醒」这一件事

### 1.3 目标用户
- 经常忘记路过药店买药的人群
- 到了超市才想起忘带购物清单的人群
- 需要多地点的服务人员（快递、外卖、销售）

## 二、技术方案

### 2.1 技术栈
| 项目       | 选型                                            | 说明                            |
| ---------- | ----------------------------------------------- | ------------------------------- |
| 跨平台框架 | Flutter                                         | 先做Android，iOS后续复用95%代码 |
| 定位服务   | `geolocator` + `flutter_background_geolocation` | 地理围栏核心                    |
| 本地存储   | `sqflite`                                       | 提醒规则存在本地，不上云        |
| 通知推送   | `flutter_local_notifications`                   | 本地通知                        |
| 地图选点   | `flutter_map` + 高德/OSM瓦片                    | 或 `amap_flutter_map`           |
| 权限管理   | `permission_handler`                            | 统一处理定位权限                |

- **最低支持**：Android 8.0 (API 26)
- **目标SDK**：Android 14 (API 34)
- **必需权限**：精确定位、后台定位、通知

## 三、功能需求

### 3.1 用户流程 — 核心操作路径

```
打开App → 地图展示当前位置 → 长按/点击添加提醒
    → 拖动围栏调整半径 → 输入提醒内容 → 保存
→ 离开App（可后台）
→ 进入围栏区域 → 收到通知提醒
→ 点击通知 → 打开App查看详情 → 可标记完成/删除
```

## 四、非功能需求

### 4.1 性能要求
| 指标         | 目标                          |
| ------------ | ----------------------------- |
| App启动时间  | < 2秒（冷启动）               |
| 围栏触发延迟 | 进入围栏后30秒内弹出通知      |
| 后台耗电     | 24小时后台监控耗电 < 5%       |
| 数据库查询   | 100条提醒下，列表渲染 < 0.5秒 |

### 4.2 兼容性要求
| 厂商      | 测试机型       | 关键问题                                |
| --------- | -------------- | --------------------------------------- |
| 小米      | Redmi Note系列 | 需引导用户开启「自启动」+「无限制省电」 |
| 华为      | P/Mate系列     | 需引导加入「忽略电池优化」白名单        |
| OPPO/vivo | R系列/X系列    | 后台定位需「允许关联启动」              |
| 三星      | Galaxy系列     | 相对友好，接近原生                      |

### 4.3 隐私与安全
- 所有位置数据只存储在手机本地，不上传任何服务器
- 申请权限时必须说明用途（如：「我们需要后台定位来检测您是否进入提醒区域」）
- 导出数据时不包含设备标识符
