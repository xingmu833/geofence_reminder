# 临场记

临场记是一款基于地理围栏的位置提醒应用。你可以提前设置目标地点，应用会在你从目标地点外进入指定范围时，通过通知或强提醒提醒你处理对应事项。

适合到店购物、到站下车、到公司处理任务、到家提醒取物等需要“到了某个地方再提醒”的场景。

## 主要功能

- 位置提醒：选择目标地点和围栏半径，到达范围后自动提醒。
- 后台监听：支持后台定位和地理围栏监听，应用退到后台后仍可判断是否到达。
- 普通通知与强提醒：可选择普通通知提醒，也可使用更醒目的强提醒。
- 提醒次数控制：支持每次进入、仅此一次、每天指定次数。
- 防误触发：创建提醒时如果人已经在目标地点内，不会立刻触发；离开后再次进入才会提醒。
- 自定义铃声：强提醒支持内置铃声，也支持选择本地音频。
- 回收站：删除的提醒会先进入回收站，可查看、还原或永久删除。
- 权限引导：提供定位、后台定位、通知、电池优化和国产 Android 后台设置提示。

## 使用说明

1. 首次打开应用后，按提示授予定位、后台定位和通知权限。
2. 在首页新增提醒，搜索或选择目标地点。
3. 设置围栏半径、提醒内容、提醒方式和提醒次数。
4. 保存后，当你从目标地点外进入目标地点范围时，应用会自动提醒。

## 权限与隐私

临场记需要定位、后台定位和通知权限来完成位置提醒。位置数据仅用于本机提醒逻辑，当前项目不包含账号同步或云端上传功能。

为了提高锁屏和后台状态下的触发稳定性，小米、红米、OPPO、vivo、华为等机型建议手动开启自启动、后台定位、通知权限和省电无限制。
